Extracted visual language:

- Warm off-white application background with white surfaces.
- Deep teal primary actions, amber warnings, red emergency states, green success/reunited states, purple match states.
- Compact operational cards with 8px radius, fine borders, and restrained shadows.
- Status badges combine color, label, and small shape/icon treatment.
- Forms are dense, bordered, and label-first.
- Privacy banners and panels are prominent, calm, and explicit.
- Desktop dashboard layouts use persistent side navigation.
- Mobile workflows use single-column cards, sticky actions, and bottom navigation/sheets.

Phase 0 implements the design system as semantic CSS/Tailwind tokens and foundational Angular components. Reference boards are not embedded or used as raster UI.
